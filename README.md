# Febor Frontend Asociados

Portal web para la gestión de asociados de **Febor** — Entidad Cooperativa de Ahorro y Crédito.

## Stack

- Angular 21
- Bootstrap 5
- TypeScript
- JWT Authentication

## Instalación

```bash
npm install --legacy-peer-deps
```

## Desarrollo

```bash
npm start
# http://localhost:4200
```

## Build producción

```bash
npm run build-prod
```

## Lint / Formato

```bash
npm run lint
npm run lint:fix
npm run prettier
```
